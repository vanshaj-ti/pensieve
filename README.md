# Pensieve

A local-first pipeline that mines your Claude Code session transcripts for daily insight briefs — see [docs/product/00-overview.md](docs/product/00-overview.md) for the full picture. This run is a scaffold only: schema, types, config loading, and a CLI stub; ingestion, chunking, extraction, and brief-writing land in later runs.

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

`pensieve analyze` is currently a stub — it logs `not yet implemented` and exits 0.
