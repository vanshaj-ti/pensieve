# Initial Product Scope

The first version of Pensieve should remain narrow.

## Inputs

* Cursor chat history;
* Claude Code logs;
* terminal history;
* optional Git metadata.

## Processing

* local or privacy-preserving inference;
* session grouping;
* decision extraction;
* insight classification;
* friction detection;
* idea detection.

## Outputs

* daily insight feed;
* weekly markdown brief;
* searchable decision archive;
* editable contribution ledger;
* private idea sandbox.

## Explicit Non-Goals

The initial product should not become:

* an employee productivity score;
* a manager analytics dashboard;
* a screen-recording tool;
* a time-tracking application;
* a generic meeting-notes product;
* a replacement for GitHub or Jira;
* a full developer-observability platform.
