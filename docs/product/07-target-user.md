# Target User

The initial target user is an experienced software engineer who:

* works heavily with AI coding tools;
* frequently uses Cursor, Claude Code, or terminal-based agents;
* cares about improving personal leverage;
* performs architectural or product-level reasoning;
* struggles to explain the full value of their contribution;
* and is uncomfortable with centralized productivity surveillance.

Potential early adopters include:

* senior engineers;
* staff and principal engineers;
* technical founders;
* engineering leads;
* AI-native product engineers;
* consultants and independent builders.
